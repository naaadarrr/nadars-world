# 如何在其他项目中使用 macOS X 组件

## 📋 完整步骤指南

### 第一步：准备你的 React 项目

假设你有一个 React 项目，项目结构如下：

```
my-react-app/
├── src/
│   ├── App.tsx
│   ├── index.tsx
│   └── index.css
├── public/
├── package.json
└── ...
```

### 第二步：复制组件包

将整个 `macosx-components` 文件夹复制到你的项目根目录：

```bash
# 方法1: 使用命令行复制
cp -r /path/to/ryos/macosx-components /path/to/my-react-app/

# 方法2: 手动复制
# 直接将 macosx-components 文件夹拖拽到你的项目根目录
```

复制后你的项目结构应该是：

```
my-react-app/
├── src/
│   ├── App.tsx
│   ├── index.tsx
│   └── index.css
├── macosx-components/     ← 新复制的组件包
│   ├── components/
│   ├── styles/
│   ├── index.ts
│   └── ...
├── public/
├── package.json
└── ...
```

### 第三步：安装依赖

在你的项目根目录运行：

```bash
npm install framer-motion zustand
```

或者使用 yarn：

```bash
yarn add framer-motion zustand
```

### 第四步：导入样式

在你的 `src/index.tsx` 或 `src/App.tsx` 文件顶部添加样式导入：

```tsx
// src/App.tsx
import React from 'react';
import './macosx-components/styles/macos.css'; // ← 添加这行

function App() {
  return <div className="App">{/* 你的应用内容 */}</div>;
}

export default App;
```

### 第五步：使用组件

现在你可以在任何组件中使用 macOS X 组件了：

#### 示例 1: 基础窗口

```tsx
// src/App.tsx
import React from 'react';
import './macosx-components/styles/macos.css';
import { MacOSWindow, MacOSThemeProvider } from './macosx-components';

function App() {
  return (
    <MacOSThemeProvider>
      <div className="min-h-screen bg-gradient-to-br from-blue-400 to-purple-600 p-8">
        <MacOSWindow
          title="我的应用"
          onClose={() => console.log('窗口关闭')}
          onMinimize={() => console.log('窗口最小化')}
          onMaximize={() => console.log('窗口最大化')}
          isForeground={true}
        >
          <div className="p-6">
            <h1 className="text-2xl font-bold mb-4">欢迎使用 macOS 风格窗口！</h1>
            <p className="text-gray-600">这是一个完全可拖拽和调整大小的窗口。</p>
            <button className="mt-4 px-4 py-2 bg-blue-500 text-white rounded hover:bg-blue-600">
              点击我
            </button>
          </div>
        </MacOSWindow>
      </div>
    </MacOSThemeProvider>
  );
}

export default App;
```

#### 示例 2: 聊天应用

```tsx
// src/ChatDemo.tsx
import React, { useState } from 'react';
import { ChatApp, MacOSThemeProvider, ChatMessage } from './macosx-components';

function ChatDemo() {
  const [messages, setMessages] = useState<ChatMessage[]>([
    {
      id: '1',
      content: '你好！我是 AI 助手，有什么可以帮你的吗？',
      role: 'assistant',
      username: 'AI助手',
      createdAt: new Date(),
    },
  ]);

  const handleSendMessage = async (messageText: string) => {
    // 添加用户消息
    const userMessage: ChatMessage = {
      id: `user-${Date.now()}`,
      content: messageText,
      role: 'user',
      username: '我',
      createdAt: new Date(),
    };
    setMessages((prev) => [...prev, userMessage]);

    // 模拟 AI 回复
    setTimeout(() => {
      const aiMessage: ChatMessage = {
        id: `ai-${Date.now()}`,
        content: `我收到了你的消息："${messageText}"。这是一个自动回复！`,
        role: 'assistant',
        username: 'AI助手',
        createdAt: new Date(),
      };
      setMessages((prev) => [...prev, aiMessage]);
    }, 1000);
  };

  return (
    <MacOSThemeProvider>
      <div className="min-h-screen bg-gradient-to-br from-green-400 to-blue-600 p-8">
        <ChatApp
          title="AI 聊天助手"
          onClose={() => console.log('聊天关闭')}
          currentUserId="我"
          initialMessages={messages}
          onSendMessage={handleSendMessage}
          showNudgeButton={true}
          showVoiceButton={false}
          emptyStateMessage="开始和 AI 助手聊天吧！"
        />
      </div>
    </MacOSThemeProvider>
  );
}

export default ChatDemo;
```

#### 示例 3: 多个窗口

```tsx
// src/MultiWindowDemo.tsx
import React, { useState } from 'react';
import { MacOSWindow, MacOSThemeProvider } from './macosx-components';

function MultiWindowDemo() {
  const [windows, setWindows] = useState([
    { id: 1, title: '计算器', x: 100, y: 100, isForeground: true },
    { id: 2, title: '记事本', x: 200, y: 150, isForeground: false },
    { id: 3, title: '设置', x: 300, y: 200, isForeground: false },
  ]);

  const bringToForeground = (id: number) => {
    setWindows((prev) =>
      prev.map((window) => ({
        ...window,
        isForeground: window.id === id,
      })),
    );
  };

  const closeWindow = (id: number) => {
    setWindows((prev) => prev.filter((window) => window.id !== id));
  };

  return (
    <MacOSThemeProvider>
      <div className="min-h-screen bg-gradient-to-br from-purple-400 to-pink-600 p-8">
        {windows.map((window) => (
          <div
            key={window.id}
            style={{
              position: 'absolute',
              left: window.x,
              top: window.y,
              zIndex: window.isForeground ? 10 : 1,
            }}
            onClick={() => bringToForeground(window.id)}
          >
            <MacOSWindow
              title={window.title}
              onClose={() => closeWindow(window.id)}
              isForeground={window.isForeground}
              windowConstraints={{
                minWidth: 300,
                minHeight: 200,
              }}
            >
              <div className="p-6">
                <h2 className="text-xl font-semibold mb-4">{window.title}</h2>
                <p className="text-gray-600 mb-4">这是 {window.title} 窗口的内容。</p>
                <p className="text-sm text-gray-500">
                  窗口 ID: {window.id}
                  <br />
                  前台状态: {window.isForeground ? '是' : '否'}
                </p>
              </div>
            </MacOSWindow>
          </div>
        ))}
      </div>
    </MacOSThemeProvider>
  );
}

export default MultiWindowDemo;
```

### 第六步：在主应用中使用

更新你的 `src/App.tsx` 来使用这些组件：

```tsx
// src/App.tsx
import React, { useState } from 'react';
import './macosx-components/styles/macos.css';
import ChatDemo from './ChatDemo';
import MultiWindowDemo from './MultiWindowDemo';

function App() {
  const [currentDemo, setCurrentDemo] = useState<'chat' | 'windows' | 'basic'>('basic');

  const renderDemo = () => {
    switch (currentDemo) {
      case 'chat':
        return <ChatDemo />;
      case 'windows':
        return <MultiWindowDemo />;
      default:
        return (
          <div className="min-h-screen bg-gradient-to-br from-blue-400 to-purple-600 flex items-center justify-center">
            <div className="text-center text-white">
              <h1 className="text-4xl font-bold mb-8">macOS X 组件演示</h1>
              <div className="space-x-4">
                <button
                  onClick={() => setCurrentDemo('chat')}
                  className="px-6 py-3 bg-white text-blue-600 rounded-lg font-semibold hover:bg-gray-100"
                >
                  聊天演示
                </button>
                <button
                  onClick={() => setCurrentDemo('windows')}
                  className="px-6 py-3 bg-white text-blue-600 rounded-lg font-semibold hover:bg-gray-100"
                >
                  多窗口演示
                </button>
              </div>
            </div>
          </div>
        );
    }
  };

  return (
    <div className="App">
      {currentDemo !== 'basic' && (
        <button
          onClick={() => setCurrentDemo('basic')}
          className="fixed top-4 left-4 z-50 px-4 py-2 bg-black text-white rounded hover:bg-gray-800"
        >
          返回主页
        </button>
      )}
      {renderDemo()}
    </div>
  );
}

export default App;
```

## 🎯 关键要点

1. **文件路径**: 组件导入路径是 `'./macosx-components'`（相对于你的源文件）
2. **样式导入**: 必须导入 `'./macosx-components/styles/macos.css'`
3. **主题提供者**: 所有组件都需要包裹在 `<MacOSThemeProvider>` 中
4. **依赖**: 确保安装了 `framer-motion` 和 `zustand`

## 🔧 常见问题

### Q: 样式没有生效怎么办？

A: 确保正确导入了 CSS 文件：`import './macosx-components/styles/macos.css';`

### Q: 组件找不到怎么办？

A: 检查文件路径是否正确，确保 `macosx-components` 文件夹在项目根目录

### Q: TypeScript 报错怎么办？

A: 确保安装了类型依赖，或者在 `tsconfig.json` 中添加路径映射

### Q: 如何自定义主题？

A: 可以创建自定义主题对象传递给 `MacOSThemeProvider`
