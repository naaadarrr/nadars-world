## 🚀 使用方法

1. **启动项目**：运行开发服务器
2. **导航到Video页面**：点击首页的"Video"文件夹
3. **播放宣传片**：
   - 点击视频缩略图，或
   - 点击"🎬 播放宣传片"按钮
4. **全屏观看**：视频将在模态框中全屏播放
5. **关闭视频**：点击右上角的"×"按钮或点击背景区域

## 📋 技术实现

### 组件结构
```
VideoPage
├── 返回按钮
├── 页面标题
├── AnyFit宣传片区域
│   ├── 标题和描述
│   ├── 视频预览缩略图
```

### 关键功能
- **状态管理**：使用React hooks管理播放状态
- **视频控制**：ref引用控制视频播放/暂停
- **模态框**：全屏覆盖层显示视频
- **事件处理**：点击、悬停、键盘事件
